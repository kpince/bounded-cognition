# Paper Outline

Working title:

Bounded, Reviewable, Externalized Cognition: A Lineage Framework for Adaptive AI Agents Without Unrestricted Self-Modification

## 1. Introduction

The adaptation problem in long-lived AI agents.

## 2. Failure Modes

- Static agents
- Raw memory accumulation
- Opaque model updates
- Unrestricted self-modification

## 3. The Middle Layer

Bounded, reviewable, externalized cognition.

## 4. Lineage and Descendants

How reusable cognition is preserved and selectively inherited.

## 5. Local Adaptation

Situated pressure, environment, and scope.

## 6. Cognitive Pull Requests

A protocol for reviewed upstream learning.

## 7. Sleep-Cycle Consolidation

Offline review and candidate generation without auto-merge.

## 8. Evaluation and Validation Gates

Human, expert, regression, and behavioral gates.

## 9. Prototype as a Hermes Skill Set

Implementation as installable skills, templates, examples, and ledgers.

## 10. Risks and Limitations

Bureaucracy, false confidence, over-scoping, under-automation, and governance overhead.

## 11. Conclusion

Safe agent learning as externalized, inspectable, selectively inheritable cognition.
